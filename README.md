# react-native-perms
React Native package for manage permissions on ios and android

## Supported permissions and native managers
- [x] Location (Always and When In Use)
- [x] Notifications (Only on iOS)
- [x] Camera
- [ ] Gallery
- [ ] Photos
- [ ] Bluetooth
- [ ] Contacts
- [ ] Reminders
- [ ] Events
- [x] Motion
- [ ] Microphone
- [ ] MediaLibrary
- [ ] SpeechRecognizer
- [ ] Siri
